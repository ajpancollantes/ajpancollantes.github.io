Complex Exponential Visualization

Visualizing the mapping of $f(z) = e^z$

The colored line in the Domain (z-plane) represents the set of complex numbers $z = x + ib$. As you move the slider, type a value, or use arrow keys, the value of $b$ and the color changes. The Codomain (w-plane) accumulates the resulting rays ($e^z = e^x \cdot e^{ib}$), showing how the exponential function maps horizontal lines to rays from the origin.
