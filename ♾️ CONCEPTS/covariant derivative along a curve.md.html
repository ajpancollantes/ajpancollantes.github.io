Covariant derivative along a curve

Given a covariant derivative operator $\nabla$ in a manifold $M$, if a vector field $X(t)$ is only defined along a curve $c$, it is shown in @malament2012topics page 55 that there exist an operator $D_c$ such that if $X$ were the restriction to $c$ of a vector field $\tilde{X}$ defined on an open set then $(D_c X)(t)=(\nabla_{c'(t)}\tilde{X} )(c(t))$.
