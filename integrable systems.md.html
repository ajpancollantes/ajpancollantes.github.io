Integrable system

It is common knowledge that every Hamiltonian system is locally integrable (away from singular points of the Hamiltonian), meaning that, in a neighborhood of each point of the $2n$-dimensional symplectic manifold on which the Hamiltonian vector field is defined, it is possible to find $n$ first integral#2. First integral of a vector fields of motion in involution. Moreover, they are, locally, superintegrable systems, in the sense that they have $(2n-1)$-first integral of motions.

The question of integrability and superintegrability it is interesting, then, if we:

  • make it a global question, or

  • still in a locall set up, we restrict the class of "interesting" first integrals, for example, to polynomials.
