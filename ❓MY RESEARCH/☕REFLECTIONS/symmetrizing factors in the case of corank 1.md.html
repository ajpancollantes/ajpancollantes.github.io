Symmetrizing factor in the case of corank 1

(xournal 195)

Lemma

Given a closed 1-form $\omega$ on an open set $U\subset \mathbb R^n$, a vector field $V$ is a symmetry of the Pfaffian system $\mathcal S (\{\omega\})$ if and only if $V\,\lrcorner\,\omega$ is a first integral of $\mathcal S (\{\omega\})$.

$\blacksquare$

Proof

The function $V\,\lrcorner\,\omega$ is a first integral if and only if $d(V\,\lrcorner\,\omega)\wedge \omega=0$. Taking into account Cartan formula this is equivalent to

$$ (\mathcal L _V \omega-V\lrcorner \,d\omega)\wedge \omega=0 $$ $$ \mathcal L _V \omega\wedge \omega=0. $$

And this is true if and only if $\mathcal L _V \omega=\beta \omega$ for some function $\beta$, i.e., $V$ is a symmetry of the Pfaffian system

$\blacksquare$

From here we can conclude several interesting facts:

Proposition 1

Given a Frobenius integrable 1-form $\omega$ and a vector field $X\notin \mathcal S(\{\omega\})^{\perp}$ then a function $f$ satisfies $fX$ is a symmetry of $\mathcal S(\{\omega\})$ (is a symmetrizing factor) if and only if $\mu=\dfrac{1}{f X\lrcorner\, \omega}$ is an integrating factor of $\omega$.

$\blacksquare$

Proof

We know that there exist an integrating factor $\tilde{\mu}$, that is, a function such that $\tilde{\mu}\omega$ is closed. Taking into account that $\mathcal S (\{\tilde{\mu}\omega\})$=$\mathcal S (\{\omega\})$, and according to the Lemma above, $fX$ is a symmetry of $\mathcal S (\{\omega\})$ if and only if $fX\lrcorner\,\tilde{\mu}\omega$ is a first integral. Now observe that

$$ fX\lrcorner\,\tilde{\mu}\omega=\dfrac{\tilde{\mu}}{\frac{1}{fX\lrcorner\,\omega}} $$

and this is a first integral if and only if $\dfrac{1}{fX\lrcorner\,\omega}$ is an integrating factor.

$\blacksquare$

Proposition 2

Given a symmetry $V$, then $1/V\lrcorner\,\omega$ is an integrating factor.

$\blacksquare$

Proof

Take $f=1$ in Proposition 1.

$\blacksquare$

Proposition 3

Given a Frobenius integrable 1-form, a smooth function $\mu$ is an integrating factor if there exists a symmetry $V$ such that $\mu=1/V\lrcorner\,\omega$.

$\blacksquare$

Proof

Take any $X$ such that $X\lrcorner\,\omega\neq 0$. Since $\mu$ is an integrating factor, by Proposition 1 $f=\dfrac{1}{\mu X\lrcorner\, \omega}$ is a symmetrizing factor for $X$, so $V=fX$ is a symmetry. It is easy to check that

$$ \mu=\dfrac{1}{V\lrcorner\,\omega}. $$

$\blacksquare$
