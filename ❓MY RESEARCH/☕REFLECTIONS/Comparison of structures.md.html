Comparison of structures

A finite dimensional symmetry algebra of an ODE with vector field $A\in \mathfrak{X}(J^{m-1})$ is an involutive distribution, so it defines integral manifolds which are, in some sense, transversal to $A$. The coefficients of the Lie brackets are constants. But suppose a weaker structure, in which we allow functional coefficients, and let's call it a "symmetry module" for an ODE. This structure also defines integral manifolds, because the vector fields are an involutive system. In both cases, we can reduce the ODE by collapsing these integral manifolds to points.

We have the following relationships:

  • If we mix the vector fields of a symmetry module by means of linear combinations with functional coefficients we obtain vector fields which are no longer symmetries of the ODE, but define the same integral manifolds. So they still can be used to reduce the ODE. They are sigma-prolongued vector fields and, in particular, a sigma-symmetry.

  • If we had only a single symmetry to start with, we could still construct a "linear combination" of itself with functional coefficients. This is a generalized Cinf-symmetry ODE.

  • On the other hand, we can consider not Lie point symmetrys but symmetry of a distribution. Within this approach you look for an "increasing sequence of distributions", starting from $\{A\}$, and every new vector field is a symmetry of the previous distribution. You use this symmetries, one by one and in reverse order, to identify together all the leaves of the associated distributions. Then repeat again. If we use cinf-symmetry of distribution instead of symmetry of a distribution, we get a cinf-structure.

  • CONJETURE: Finally, it may happen that the vector fields in the symmetry module are "well ordered", in the sense that if we use the first one to reduce, the other vector fields project in symmetries of the reduced ODE (solvable symmetry algebra). I think that this property can be satisfied by some sigma-symmetrys, if we require some conditions to the matrix $\sigma$.
